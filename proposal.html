Project Background

Boolean Query is one of the most important types of query in an Information Retrieval System. Lucene has done several optimizations for that. The kernel modular to implement Boolean Query in lucene is Scorer, which can not only score a document, but also provide methods for collecting Hit Documents. There are mainly two Scorers to deal with Boolean Query on current trunk.

BooleanScorer vs. BooleanScorer2
